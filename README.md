
 🌦️ Weather Application

A responsive Weather Application built using HTML, CSS, JavaScript, and OpenWeather API.

The application allows users to search weather information for any city and also supports current location-based weather updates.

 🚀 Features

* Search weather by city name
* Current location weather detection
* Real-time weather information
* Temperature and weather condition display
* Loading and error handling states
* Responsive design for mobile and desktop devices

 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeather API

 What I Learned

* API Integration
* Asynchronous JavaScript
* Fetch API
* DOM Manipulation
* Error Handling
* Responsive Web Design
* Geolocation API
