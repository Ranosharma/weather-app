# Weather App

A responsive weather application that provides real-time weather information for any city using a weather API. The application is designed with a clean interface, making it easy to check current weather conditions instantly.

---

## Preview

![Weather App Banner](https://img.shields.io/badge/Weather-App-4FC3F7?style=for-the-badge&logo=weather&logoColor=white)

---

## Features

- Search weather by city name
- Real-time temperature updates
- Displays weather condition with icon
- Shows humidity
- Wind speed information
- Responsive design for desktop and mobile
- User-friendly interface
- Fast API integration

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
- Fetch API

## How It Works

1. Enter the name of any city.
2. Click the Search button.
3. The application sends a request to the weather API.
4. Current weather information is fetched.
5. Weather details are displayed instantly.
